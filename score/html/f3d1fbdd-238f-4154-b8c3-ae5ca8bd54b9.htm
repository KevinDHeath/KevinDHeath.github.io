      Sample games

      These are examples of what the user interface would look like. The pages have been generated using a test feature that has logic to act for each player in determining how their cards are played. This makes decisions only on what is visible to all players and the cards in their own hand.

      • Game over
        Always shows the cards in hand for all players at the end of the game, and the score cards for each hand played in the game.

      • In progress
        Only shows the specific players cards in hand, and the score cards for any previous hands completed in the game. A button is provided to simulate what the player would see when they display there own details.

      • Long game
        This is another completed game, however the target was set to $500,000 which resulted in more hands having to be played.
