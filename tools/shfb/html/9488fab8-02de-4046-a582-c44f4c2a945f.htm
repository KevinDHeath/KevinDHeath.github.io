Components

This site contains technical information about the kdheath Windows Presentation Foundation (WPF) packages available on NuGet.

Components

Attributes
Package: Wpf.Resources

Contains classes for custom validation attributes.

Commands
Package: Wpf.Resources

Contains classes that implement the ICommand interface which enables a user to interact with the business logic.

Controls
Package: Wpf.Controls

Contains classes to create elements, known as controls, that enables a user to interact with an application.

Converters
Package: Wpf.Resources

Contains classes to convert binding source data to target data.

Rules
Package: Wpf.Resources

Contains classes to validate the WPF user interface elements.

Themes
Package: Wpf.Resources

Contains Windows Desktop resources which can be includes in WPF .NET projects.

Note: The documentation for the other packages can be found at .NET Helper Packages.
