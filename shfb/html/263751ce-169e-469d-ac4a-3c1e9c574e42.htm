      DatePicker Styles

      These styles relate to the Custom TargetType="{x:Type cc:DatePicker} control class.

      The control provides a dependency property to determine if error messages are shown below the control.

      controlDatePickerStyle

      This style is automatically assigned to the Date Picker custom control:

      XAML
      <cc:DatePicker/>

      Default properties

      • IsErrorShown = "False"

      • SelectedDateFormat = "Long"

      IsErrorShown property

      Example of assigning the IsErrorShown style to a Date Picker custom control:

      XAML
      <cc:DatePicker IsErrorShown="True"/>

      Notes

      1. Use this property to show error messages in a text block attached to the date picker. This allows for fields to automatically adjust their position on the form:

      2. To allow the whole form (i.e., Window) to adjust its size, the property SizeToContent="Height" must be set.
