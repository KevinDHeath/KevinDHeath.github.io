Introduction

This site contains technical information about the kdheath .NET Windows Presentation Foundation packages available on NuGet.

Components

Wpf.Commands
Package: Wpf.Resources		Contains classes that implement the ICommand interface which enables a user to interact with the business logic.
Wpf.Controls
Package: Wpf.Controls		Contains classes to create elements, known as controls, that enables a user to interact with an application.
Wpf.Converters
Package: Wpf.Resources		Contains classes to convert binding source data to target data.
Wpf.Rules
Package: Wpf.Resources		Contains classes to validate the WPF user interface elements.
Wpf.Themes
Package: Wpf.Resources		Contains Windows Desktop resources which can be includes in WPF .NET projects.
