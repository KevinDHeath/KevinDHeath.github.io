      PasswordBox Styles

      These styles relate to the Windows Desktop TargetType="{x:Type PasswordBox} control class.

      Example of adding the resources to an App.xaml file:

      XAML
      <ResourceDictionary>
  <ResourceDictionary.MergedDictionaries>
    <ResourceDictionary Source="pack://application:,,,/Common.Wpf.Resources;component/Themes/PasswordBox.xaml"/>
  </ResourceDictionary.MergedDictionaries>
</ResourceDictionary>

      commonPasswordBoxEditTemplate

      Example of assigning the template to a custom Password Box style:

      XAML
      <Style x:Key="appPasswordBoxStyle" TargetType="{x:Type Password}">
  <Setter Property="Template" Value="{StaticResource commonPasswordBoxEditTemplate}"/>
</Style>

      Notes

      • Use this template for editable password boxes with rounded corners.

      commonPasswordBoxStyle

      Example of assigning the style to a Password Box control:

      XAML
      <PasswordBox x:Name="pbName" Style="{StaticResource commonPasswordBoxStyle}"/>

      Default properties

      • Template = "{StaticResource commonPasswordBoxEditTemplate}"

      • Validation.ErrorTemplate = "{x:Null}"

      commonPasswordBoxWithErrorsStyle

      Example of assigning the style to a Password Box control:

      XAML
      <PasswordBox x:Name="pbName" Style="{StaticResource commonPasswordBoxWithErrorsStyle}"/>

      Default properties

      • Template = "{StaticResource commonPasswordBoxWithErrorsTemplate}"

      • Validation.ErrorTemplate = "StaticResource commonErrorItemTemplate"

      Notes

      1. Use this style to show error messages in a text block attached to the password box. This allows for fields to automatically adjust their position on the form:

      2. To allow the whole form (i.e., Window) to adjust its size, the property SizeToContent="Height" must be set.
