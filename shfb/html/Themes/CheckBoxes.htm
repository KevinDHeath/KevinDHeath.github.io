CheckBox Styles

These styles relate to the Windows Desktop TargetType="{x:Type CheckBox} control class

Example of adding the resources to an App.xaml file:

XAML 
<ResourceDictionary>
  <ResourceDictionary.MergedDictionaries>
    <ResourceDictionary Source="pack://application:,,,/Common.Wpf.Resources;component/Themes/CheckBox.xaml"/>
  </ResourceDictionary.MergedDictionaries>
</ResourceDictionary>

commonCheckBoxStyle

Example of assigning the style to a Check Box control:

XAML 
<CheckBox x:Name="cbBackup" Style="{StaticResource commonCheckBoxStyle}"/>

Default properties

  • BorderThickness = "1"

  • Margin = "3"

  • VerticalAlignment = "Center"
