      ListView Styles

      These styles relate to the Windows Desktop TargetType="{x:Type ListView} and TargetType="{x:Type GridViewColumnHeader} control classes.

      Example of adding the resources to an App.xaml file:

      XAML
      <ResourceDictionary>
  <ResourceDictionary.MergedDictionaries>
    <ResourceDictionary Source="pack://application:,,,/Common.Wpf.Resources;component/Themes/ListView.xaml"/>
  </ResourceDictionary.MergedDictionaries>
</ResourceDictionary>

      commonListViewStyle

      Example of assigning the style to a List View control:

      XAML
      <ListView x:Name="UsersList" Style="{StaticResource commonListViewStyle}"/>

      Default properties

      1. Background = "GhostWhite"

      2. SelectionMode = "Single"

      commonGridColumnHeaderStyle

      Example of assigning the style to a Grid View Column Header control:

      XAML
      <GridViewColumn HeaderContainerStyle="{StaticResource commonGridColumnHeaderStyle}"/>

      Default properties

      • FontWeight = "Bold"

      • HorizontalContentAlignment = "Left"

      • Margin = "5,0,0,0"
